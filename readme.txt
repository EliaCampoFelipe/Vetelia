
Instalació:

Copiar els fitxers en una carpeta (o extreure'ls del .zip)
Cal connexió a internet per a:
    Instal·lar python 3.11
    Instal·lar Pillow per a Python executant en una consola l'ordre pip install Pillow
    Instal·lar sqlite3 des de la pagina oficial https://www.sqlite.org/download.html


Execució:

Obrir una consola i dins de la carpeta on s'hagin copiat els fitxers, executar l'ordre
python vetelia.py
o
python vetelia.py castellano si es vol en castellà
